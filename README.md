BestPractice 最佳实践
============

# 项目列表
## Maven In Action (Maven实战 许晓斌著)
	针对书中多个最佳实践进行练习，如maven自动构建、聚合和继承、资源过滤、多种依赖处理
	其中对于junit的用法约定、随即验证码图片生成是不错的体验

## ItxdlInAction (国内IT兄弟连培训教育)
	根据视频提供的源码、工具、文档进行针对性的查疑补漏
	包括java se、java ee、android三方面的

## MyJpa (根据项目需要针对JPA中对领域模型进行数据库重建的模块开发和研究)
	jpaCreateDBDemo 使用hibernate组件库生成/更新数据库
		使用javax.persistence.EntityManagerFactory 这个原生的实体管理工厂来实现
	SpringJpaDemo 使用spring相关组件生成/更新数据库，并测试了骆驼命名法转换为下划线加小写的实现。
		使用LocalContainerEntityManagerFactoryBean 和hibernate的org.hibernate.cfg.ImprovedNamingStrategy 命名策略来实现
